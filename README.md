# Jinka Gowri | Personal Portfolio Website

A modern, responsive, and interactive portfolio website developed using React.js to showcase my technical skills, projects, certifications, achievements, and professional profile.

The portfolio is designed with a clean UI, smooth animations, responsive layouts, and an integrated contact system to create an engaging user experience.

## Live Website

🌐 https://portfolio-sandy-eight-89.vercel.app

## GitHub Repository

📁 https://github.com/Gowri-jinka/Portfolio

---

## Project Overview

This portfolio serves as my digital professional profile and highlights:

- Technical skills and technologies
- Full-stack and analytics projects
- Certifications and achievements
- Resume and academic details
- Competitive programming profiles
- Contact integration using EmailJS
- Responsive design for desktop and mobile devices

---

## Key Features

- Interactive Hero Section
- Smooth Scroll Navigation
- Animated UI using Framer Motion
- Responsive Design
- Project Showcase Cards
- Skills & Education Sections
- Certificates and Activities
- EmailJS Contact Form Integration
- Social Media Links
- Scroll-to-Top Feature
